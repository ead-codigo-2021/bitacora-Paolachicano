Cuarta Clase

Variables css: permite poder guardar un valor y reutilizarlo, tambien se pueden sobre escribir estas variables en un contexto más específico.
Se escriben utilizando dos guiones

font-face con google fonts
se puede hacer a traves de link y a través de import, la más recomendada es link
Se deben tener en cuenta todos los pesos que se utilizarán, para cargarlos desde el navegador
Para utilizar una fuente descargada de algún sitio, posterior a asegurarse que se cumpla con las licencias de uso, se descarga el archivo de la tipografía, luego se construye una relación font-face parecida ala que construye google fonts, debe estar en formato otf o ttf. Este archivo debe estar alojado en la carpeta del index.html, dentro de una propia carpeta de fuentes, y llamar la tipografia con font-face pero desde la carpeta.

Se recomienda que las declaraciones de fuente esten siempre al principio del codigo css, se debe respetar el nombre de cada tipografia. Si se ocupa una tipografía estática se debe respetar cada uno de los nombres de las variantes.
Se deben declarar tambien los pesos de la tipografía, el display y el estilo.

Se pueden definir las tipografías como variables, para así realizar combinaciones tipográficas.
El uso de variables permite ordenar el uso de los distintos contenidos a utilizar, ya sea fuentes, colores, etc...

Wakamaifondue.com es una herramienta que permite analizar todas las características que tiene un archivo de fuente, tamaños, pesos, idiomas, etc...

Los tamaños tipograficos pueden estar en pixeles, porcentaje, em o rem, estos ultimos son recomendados porque son a razon de la tipografía.
Con @media screen, se puede definir el tamaño maximo de una tipografía para tamaño móvil.

Sexta Clase

Posiciones en cascada de estilos: permiten modificar desde donde se ubica un elemento, en conjunto con las propiedades top, left, right y bottom.
Las posiciones son static, relative, absolute y sticky.

Diagramación CSS:
Flexbox: Permite el ordenar cajas de contenido, tanto orden como posición. Posee contenedor flexible, elementos flexibles, eje principal, eje transversal y líneas.
justify-content: alineación de elementos en el eje principal.
align-items: alineación de elementos en el eje secundario.
align-self: el elemento se alínea segun los otros.
Flexbox permite también el controlar las proporsiones de los elementos.

Séptima Clase

Un componente es, en términos abstractos, una solución para un problema, en getbootstrap.com se pueden ver los tipos de componentes existentes.
Un framework es la posibilidad de reutilizar estas soluciones en distintos contextos.

Flexbox tiene el objetivo de darle estilo o la capacidad flexible a un contenedor pero respecto a sus contenidos.
CSS Grid: Permite distribuir y diagramar los elementos en una página. Grid se utiliza declarando display grid, al igual que flexbox, se diagrama en el eje de columnas y en el eje de filas, con grid-template-columns y grid-template-rows. A estas zonas se les otorga su tamaño o proporción con respecto a la página.
Fr es una medida propia de grid, lo cual es una fracción de pantalla que se calcula segun la cantidad de fracciones de esta.
Cada elemento se debe especificar su posición en la columna y fila correspondiente a su posición deseada. Con grid-template-areas se pueden definir areas para posicionar elementos.
Siempre es bueno que todos los elementos tengan una clase, independiente de su etiqueta.
La grilla está compuesta por líneas, las cuales están entremedio de los espacios divisorios, los tracks, que es el área delimitada por dos líneas, las celdas, delimitadas por dos líneas y columnas adyacentes y las áreas, delimitadas por 4 líneas.
Column-gap y Row-gap sirven para definir los espacios entre columnas y filas.

Novena Clase

JS y procesadores.
Con JS se pueden realizar operaciones y comparaciones de variables. Desde esto puede entregar respuestas con estos datos.
Estas operaciones se realizan a traves de funciones condicionales, como for, if, while, etc...
Un objeto es pensar en los datos de los elementos que se mostraran en pantalla, pensar un elemento y sus atributos.
Tipos de datos básicos: existen los strings, que son textos, number, que son números, boolean, que son verdadero y falso, array que es una colección de elementos de cualquier tipo, object que son una colección de propiedades de los objetos, undefined, que son datos indefinidos y null que son datos nulos.
1 == '1', entero 1 es igual a string '1' (a veces).
1 !== '1', no es igual.
El tipo de dato importa.
jQuery es una librería de javascript, que permite la realización de manipulación de documentos, manejo de eventos, animacion y ajax de manera mucho mas simple.
Para utilizarlo debe incluirse el archivo script, recomendada al final del body del archivo html.
Vue permite pensar aplicaciones, diseñar ingreso de contenidos, etc...
Preprocesar, compilar, transpilar. Un preprocesador CSS es un programa que convierte un tipo de información en hojas de estilo css. Para css existen 3 frameworks para escribir scss, {less}, Sass y stylus.
Existen extensiones de visual studio code para compilar sass, como por ejemplo live sass compiler.
En scss se pueden anidar los selectores de los elementos descendentes, lo que permite visualizar mejor la jerarquía de los elementos.
Se debe tener cuidado ya que la especificidad de los elementos puede aumentar sin darnos cuenta.
Se pueden realizar operaciones matemáticas dentro del valor de una propiedad.
Sass ofrece una larga lista de funciones que podemos utilizar para gestionar colores, hacer cálculos o programar funciones más complejas.
Una de la principales ventajas de un preprocesador es la capacidad de importar otros archivos scss o css, lo que permite aislar estilos en hojas separadas.
Los Mixins e una forma de crear funciones propias, para agrupar estilos comunes que puedan generarse a partir de variables.
Node.js permite ejecutar javascript en el servidor. Npm: Node Package Manager, es un registro de software y gestor de dependencias para node.js, permite buscar e instalar paquetes dentro de un proyecto.
